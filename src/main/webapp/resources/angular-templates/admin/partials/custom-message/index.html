Send custom message to all attendees

Write a custom message and broadcast it to all attendees

Available variables:

the syntax is defined by Mustache, with some limitations
eventName
the name of the event
fullName
Either the attendee's name of the name of the reservation owner
organizationName
The name of the organization
organizationEmail
The organization e-mail address
reservationID
The reservation identifier
reservationURL
The URL of the reservation page
